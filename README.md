# basic_chatbot
Simple Rule-Based Chatbot

Project Description

This is a simple rule-based chatbot developed in Python. The chatbot responds to basic user inputs such as greetings, questions about its name, and "How are you?". The conversation continues until the user types "bye".

Features

- Responds to greetings like Hi, Hello, Hey, and Hii.
- Answers "How are you?".
- Tells the user its name.
- Ends the conversation when the user types bye.
- Handles unknown inputs with a default response.

Technologies Used

- Python 3
- Functions
- Loops
- Conditional Statements
- Lists
- User Input

How to Run

1. Make sure Python 3 is installed.
2. Save the program as "chatbot.py".
3. Open a terminal or command prompt.
4. Run the program using:

python chatbot.py

5. Start chatting with the bot.
6. Type bye to end the conversation.

Supported Commands

- Hi
- Hello
- Hey
- Hii
- How are you
- What is your name?
- Bye
